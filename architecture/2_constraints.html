Constraints

Organizational constraints

IDConstraintsMotivation
OC1AffordableThe cost to assemble a WordClock should be low.
OC2DocumentationUse arc42 as documentation template.
OC3LanguageUse English for documentation.

Technical constraints

IDConstraintsMotivation
TC1Open-sourcePrivilege open-source solutions as software dependencies.
TC2Programming languageUse Rust as programming language.
TC3ToolsUse as much as possible the Rust ecosystem for software development and documentation.
TC4Version controlUse Git and GitHub for version control.

Electronic / Mechanical constraints

IDConstraintsMotivation
EMC1DimensionsThe electronic dimension should not exceed 190x190[mm]
EMC2Power cableThe electronic should be powered with a USB Micro-B cable.
EMC3User inputThe electronic should provide at least a push-button for interacting with the system.
EMC4LabelsThe electronic should be labeled for user with no electronic knowledge.
EMC5FixationThe electronic should provide 4x 4[mm] fixation holes.
